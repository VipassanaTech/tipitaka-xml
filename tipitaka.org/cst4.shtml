Chaṭṭha Saṅgāyana Tipiṭaka Version 4.0 (CST4)

CST4 is a desktop client for browsing and searching the Pāli Canon. It is the successor to the Chaṭṭha Saṅgāyana CD 3.0 (CSCD3).

Download for Windows

Installer updated May, 2020 with corrected Tipiṭaka text files.

System requirements: Windows 10, 8.1, 8, 7 or XP.

Depending on your version of Windows, you may need to enable or install the .NET Framework 2.0, a component of Windows.

The program installer is an MSI file. Windows XP users may need to install the Windows Installer component.

To type Roman-script Unicode Pali, download and install a Pali keyboard.

Enable complex script support in Windows XP. Go to Control Panel -> Regional and Language Options -> Languages -> Click the checkbox “Install files for complex scripts including right to left languages (including Thai)”. Then OK.

To type in Devanagari and other Indic scripts, install the keyboard drivers included with Windows. Installation instructions: Windows 10, Windows 7, 8 or 8.1.

To run the program, click the “Chattha Sangayana Tipitaka 4.0” shortcut created by the installer. The search index is generated the first time the program runs. This can take up to 15 minutes.

To report bugs, make suggestions or request additional features, please send an email to the development team at help@tipitaka.org. Your input is highly appreciated.
